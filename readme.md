# 📈 Marketing ROI Analysis with Simple Linear Regression

##  Project Summary

Businesses invest heavily in marketing, but which channel actually drives sales?

In this project, I use **Python** and **Ordinary Least Squares (OLS) Linear Regression** to identify the marketing channel with the greatest impact on sales. From data cleaning and exploratory analysis to statistical modeling and business recommendations, this project demonstrates an end-to-end data analytics workflow for solving a real-world business problem.

The final outcome is a data-driven recommendation that helps optimize marketing budget allocation and maximize Return on Investment (ROI).

---

##  Business Problem

Marketing teams often distribute advertising budgets across multiple channels without knowing which delivers the highest return.

This analysis answers the question:

> **Which marketing channel contributes most to sales, and where should a company invest more of its advertising budget?**

---

##  Project Objectives

- Clean and prepare raw marketing data for analysis.
- Perform Exploratory Data Analysis (EDA) to uncover trends and relationships.
- Identify the strongest predictor of sales.
- Develop an interpretable Simple Linear Regression model using **statsmodels**.
- Validate model assumptions through diagnostic testing.
- Translate statistical findings into actionable business insights.
- Provide an evidence-based recommendation for marketing investment.
